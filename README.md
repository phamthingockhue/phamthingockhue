* **Tech Stack:** `C#` | `.NET WinForms` | `SQL Server`
* **Architecture:** Strictly followed **3-Layer Architecture** (Presentation, Business Logic, Data Access Layers).
* **Key Mechanisms:** * Completely decoupled UI controllers from business logic rules.
  * Optimized relational database schemas with indexing, ensuring strong data integrity and preventing data anomalies.
